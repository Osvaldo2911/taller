Frameworks y librerias

En el trancurso de la mmateria de programacion web hemos visto y trabajado con html, css y java script, para realizxar las practicas realizadas en el aula de clases, una vez aprendido lo anterior pasamos a conocer frameworks y librerias que podriamos colocar y complementar en nuestras paginas o aplicaciones webs, con ese objetivo se realizaron talleres en los cuales expondriamos un frameworks o libreria asignada por el profesor, por medio de esta web presentar esas exposisiones y practicas realizadas en cada taller.


Talleres
